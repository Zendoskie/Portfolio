# Portfolio Website

Modern, futuristic personal portfolio website for John Paul C. Marquez.

## Setup

1. **Add Profile Photo**: 
   - Save your profile photo as `profile.jpg` in the `assets/images/` folder
   - The image should be square or will be cropped to a circle
   - Recommended size: 500x500px or larger

2. **Open the website**:
   - Simply open `index.html` in your browser
   - Or deploy to GitHub Pages
